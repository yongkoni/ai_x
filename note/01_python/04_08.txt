1장 파이썬 개요 및 환경 설정

1절 Jyputer Notebook 사용법
	shift+Enter : 선택된 셀이 실행되고, 다음셀로 이동
	alt+Enter : 선택된 셀이 실행되고, 다음셀이 생성된 후 다음셀로 이동
	ctrl+Enter : 선택된 셀이 실행되고, 해당셀에 있음
	Enter(셀 클릭) : 다시 편집상태로 돌아감

2절 핫키
	dd : 삭제
	a : 윗셀 추가
	b : 아렛셀 추가
	m : 마크다운셀(문서셀)로 변경
	y : 코드셀로 변경
	o : 실행결과 접기/펴기(토글가능)

3절 마크다운
	코드와 함께 문서화 할 수 있음
	다양한 문서화 문법 ex. README.md


2장 자료형과 연산자

1절 변수
	선언이 필요없고 할당과 동시에 타입이 셋팅
	문자, 숫자, _로 변수명작성
	숫자로 시작하거나 예약어는 사용불가
	다중 변수할당시 일괄 할당

2절 화면 i/o
	input()은 입력된 값을 문자로 반환
	print(출력 내용, sep=' ', end='\n')

3절 기본 자료형
	숫자형 : int, float, complex
	           int값/int값 => 실수
	           지수형 표현(큰수, 작은수) => 실수
	문자형 : str
	          문자열 연산 : +(연결) *(반복)
