3. 기본 이미지 생성
	이미지 생성 요청하기
	DALL-E 3 모델 사용 (선택 사항)
		quality : DALL-E 3 전용 옵션으로, "standard" (기본값) 또는 "hd" 값을 가짐 
			"hd"는 더 높은 이미지 품질을 제공하지만 생성 비용과 시간이 증가 
			"standard"는 빠르고 비용이 저렴하지만 품질이 다소 낮음
		style : 마찬가지로 DALL-E 3의 추가 옵션 
			"vivid"(선명하고 풍부한 색감, 기본값) 또는 "natural"(더 자연스러운 색감과 톤) 중 선택하여 이미지의 스타일 분위기를 조정 가능
			원하는 결과에 따라 어떤 스타일이 적합한지 실험 가능

OpenAI TTS API

1. 환경 설정

2. 기본 TTS 변환 : 텍스트를 음성으로 간단히 만들기
	model = "tts-1" : OpenAI의 기본 TTS 모델을 지정하며 tts-1 모델은 실시간 응답에 최적화된 모델로, 비교적 빠르게 결과를 반환
	voice = "nova" : 출력 음성의 목소리를 지정
	input = text : 변환할 텍스트 문자열을 입력으로 전달

3. 고급 기능 활용: 모델 선택, 목소리 변경, 언어 지원 등
	다양한 목소리(voice) 선택 : 남성/여성 및 톤이 다른 여러 프리셋 음성을 사용 가능
	모델 선택 : 기본 모델 tts-1 외에 고품질 모델 tts-1-hd 사용법
	다국어 지원 : 영어 이외의 언어로도 TTS가 가능 (한국어, 스페인어 등)
	음성 속도 조절 : speed 파라미터로 발음 속도를 빠르게 또는 느리게 조절

	3.1 여러 가지 음성 목소리 선택하기
		Alloy – 남성적이고 부드러운 음색
		Echo – 맑고 청아한 톤
		Fable – 동화 구연 같은 따뜻한 톤
		Nova – 친근한 여성 음색으로 추정
		Onyx – 깊고 진중한 남성 음색
		Shimmer – 밝고 생동감 있는 여성 음색

	3.2 TTS 모델 선택: 실시간 vs 고품질

	3.3 다국어 텍스트 변환하기

	3.4 발음 속도 조절하기
		speed = 1.0 : 기본 속도
		speed < 1.0 : 느린 발음
		speed > 1.0 : 빠른 발음

4. 오디오 파일 저장 및 Jupyter에서 재생하기

OpenAI Whisper API

1. 환경 설정

2. Whisper API 소개 및 기본 음성 → 텍스트 변환
	Transcriptions endpoint : 제공한 음성 파일을 해당 음성의 원어로 필사(글로 변환) 
	Translations endpoint : 제공한 음성 파일을 영어로 번역하여 텍스트로 반환

	기본 사용법: 음성 파일 필사 요청
		1) 오디오 파일 준비
		2) API 요청 구성
			file : 열어둔 오디오 파일 객체
			model : 사용할 모델 ID
		3) 응답 처리

3. 다양한 언어의 음성 인식 및 언어감지

4. 고급 기능 1 : 타임스탬프 포함 자막 및 세부 출력
	다양한 출력 형식 (response_format 옵션)
		"json" (기본값) : { "text": "..." } 형태의 JSON 응답. 추가 메타정보 없이 최종 텍스트만 포함
		"text" : 순수 텍스트로 변환 결과만 반환
		"srt" : SubRip 자막 형식으로 반환. 자막처럼 각 문장에 시간 코드가 포함
		"vtt" : WebVTT 자막 형식으로 반환.
		"verbose_json" : 인식 결과에 대한 상세 정보를 담은 JSON으로 반환. 인식된 문장별 세그먼트와 타임스탬프, 확률 등 메타데이터가 모두 포함

5. 고급 기능 2 : 실시간 음성 변환 (마이크 입력 처리)
	마이크 입력 녹음하여 전송하기
		1) 짧은 음성 녹음 : sounddevice.rec 함수를 사용
		2) 오디오 데이터를 파일로 저장
		3) 저장한 파일 전송
